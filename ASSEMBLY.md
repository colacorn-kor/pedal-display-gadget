# pedal-display-gadget — 프로토타입 조립 가이드

펌웨어 핀맵(`app_main.c`, `display_bringup.c`)에서 그대로 뽑은 배선표 + 전원 체인 +
아날로그 프론트엔드 + 단계별 브링업 절차. 부품 도착하면 위에서부터 순서대로 진행.

> J201(뮤트)·SD는 **나중 단계**다. 사유는 맨 아래 "보류 항목" 참고.
> 첫 조립으로 **디스플레이 + 버튼 + 튜너/비주얼라이저**까지 전부 동작한다.
> 튜너 검출 음역 = **30~1300Hz**(베이스 5현 저B 31Hz·드롭튜닝 포함, 리뷰 완료).
> 비주얼라이저(MONITOR)는 별개로 20Hz~20kHz.

## 1. GPIO 배선표 (ESP32-S3-DevKitC-1)

| GPIO | 역할 | 상대편 핀 | 비고 |
|------|------|-----------|------|
| 1  | BTN_UP / 상     | 택트 스위치 → GND | 내부 풀업, 외부 저항 불필요 |
| 2  | BTN_DOWN / 하   | 택트 스위치 → GND | 〃 |
| 4  | BTN_LEFT / 좌   | 택트 스위치 → GND | 〃 |
| 5  | BTN_OK / 확인   | 택트 스위치 → GND | 〃 |
| 6  | BTN_RIGHT / 우  | 택트 스위치 → GND | 〃 |
| 7  | FOOTSW        | AT1D-2M3 → GND    | 〃 (모멘터리) 숏=다음 앱 / 롱=튜너 점프 |
| 8  | LCD 백라이트   | ST7796 BLK/LED    | 직결 시도 → 어두우면/뜨거우면 2N7000 경유 |
| 9  | LCD DC        | ST7796 DC(RS)     | |
| 10 | LCD CS        | ST7796 CS         | |
| 11 | LCD MOSI      | ST7796 SDA/MOSI   | |
| 12 | LCD SCLK      | ST7796 SCK        | |
| 13 | BTN_HOME / 홈   | 택트 스위치 → GND | 내부 풀업. 숏=공통 팝업(Exit/Settings/Help/About) / 롱=즉시 나가기 |
| 14 | LCD RST       | ST7796 RST        | |
| 15 | I2S DIN       | PCM1808 OUT(DOUT) | ADC 데이터 입력 |
| 16 | I2S MCLK      | PCM1808 SCKI(SCK) | ESP32가 마스터 클럭 공급 |
| 17 | I2S BCK       | PCM1808 BCK       | |
| 18 | I2S WS/LRCK   | PCM1808 LRC(LRCK) | |
| 21 | MUTE          | J201 게이트 회로  | **나중**(J201 도착 후) |

버튼 6개(상·하·좌·우·확인·홈) + 풋스위치 = **7개**, 전부 **한쪽 GPIO ↔ 반대쪽 GND**.
눌리면 LOW. (펌웨어가 `GPIO_PULLUP_ENABLE`) 입력단은 디바운스 30ms + 방향키 오토리피트
(400ms 후 120ms 간격) + 홈/풋스위치 롱프레스(500ms) 판정을 한다.

스트래핑(0·3·45·46)·USB(19·20)·옥타 PSRAM/플래시(26~37)와 충돌 없음 — 이미 회피된 핀들.
추가 버튼 핀 1·2·13도 이 회피 규칙을 지키며, 38·48은 DevKitC-1 온보드 RGB LED와 충돌
가능해 제외했다.

## 2. 전원 체인

```
9V 어댑터
   │
ELB040202 (배럴잭→터미널)
   ├─[A]→ MP1584 IN ─→ 5.0V ─┬─→ DevKit 5V 핀  (ESP32 + 디스플레이 3.3V는 온보드 LDO)
   │                          └─→ AMS1117 IN ─→ 3.3V(클린) ─→ PCM1808 아날로그
   └─[B]→ RC 필터 ─→ 9V(클린) ─→ TL072 V+   (V− = GND, 가상GND 4.5V)
```

### ⚠ MP1584는 연결 전에 먼저 5.0V로 맞춰라
1. MP1584만 단독으로 9V 입력에 물린다(출력 아무 데도 연결 안 함).
2. 멀티미터로 OUT 측정하며 가변저항(트리머)을 돌려 **정확히 5.0V**.
3. **그다음에** DevKit 5V 핀에 연결. (이 순서 안 지키면 보드가 죽는다.)

### 레일별 정리
- **디지털 5V**: MP1584 → DevKit 5V 핀. (큰 전류=백라이트 포함, 스위처라 시원함)
- **아날로그 3.3V**: MP1584 5V → AMS1117(입력 4.5~7V 사양) → 3.3V. 스위처 리플을
  LDO가 한 번 걸러줌 → PCM1808 노이즈 플로어용 클린 레일.
- **op-amp 9V**: ELB040202에서 9V 직탭 → 직렬 100Ω + 100µF + 100nF → TL072 V+.
  디지털과 분리돼야 깨끗함. (전류 ~5mA라 강하·발열 무시)

### 공통 그라운드
모든 GND(9V 어댑터·MP1584·AMS1117·DevKit·PCM1808·TL072·잭)를 **한 점(스타 그라운드)**
에서 묶는다. 가능하면 전원 입력부 근처에서.

### USB vs 5V 동시 주의
플래싱·첫 테스트는 **USB만**으로. MP1584 5V를 DevKit에 물린 상태에서 USB도 동시에
꽂는 건 보드 전원경로에 따라 백피드 위험 → 처음엔 둘 중 하나만.

## 3. 아날로그 프론트엔드 (TL072 병렬 버퍼)

목적: 기타 신호를 **통과 톤에 영향 없이 병렬로 탭**해서 PCM1808에 넣기.
PCM1808 입력 사양 = **single-ended 3Vp-p 풀스케일**.

```
입력잭 Tip ──┬─────────────────────────────→ 출력잭 Tip (패스스루, 직결)
            │                                     └─(나중) J201 뮤트 션트
            │  ┌──────── TL072 (1/2) 비반전 버퍼+게인 ────────┐
            └─ 1µF ─┬─ [+in]                                  │
                    1MΩ→Vref(4.5V)         [out] ─ 1µF ─→ PCM1808 IN(L)
                              [−in]─Rg(10k)─GND
                                  └─Rf(15k)─[out]      게인 = 1+Rf/Rg ≈ 2.5×
```

- **가상 그라운드 Vref=4.5V**: 9V에서 10k+10k 분배 + 10~100µF + 100nF.
  TL072 **나머지 반쪽**으로 Vref를 버퍼하면 더 안정적(듀얼이라 딱 맞음).
- **게인**: 3Vp-p는 *최대*다. 피크에서 클리핑 안 나게 처음엔 **×2 정도로 보수적**으로
  시작 → Monitor 화면 보고 저항 바꿔 튜닝. (R 키트로 값 스왑)
- **입력 바이어스**: PCM1808 입력단에 온보드 바이어스가 있으면 1µF AC 커플만,
  없으면 입력 핀을 (아날로그 VCC/2)로 분배 바이어스. → **모듈 상세 확인 필요**.
- 안티앨리어싱은 PCM1808 내장 아날로그 LPF + 64x 오버샘플링이 처리 → 외부 필터 불필요.
- **⚠ 저역 통과 (튜너 30Hz 대응)**: 튜너가 베이스 저B(31Hz)·드롭튜닝까지 커버하므로
  AC 커플링의 하이패스 코너가 **30Hz를 깎으면 안 된다**. 1µF + 1MΩ = 코너 0.16Hz라
  여유 충분 → **이 캡을 작은 값으로 줄이지 말 것.** 출력 쪽 1µF도 PCM1808 입력
  임피던스와의 코너가 ~15Hz 이하인지 확인(모듈 바이어스망에 따라 달라짐).
  PCM1808 내장 HPF는 0.91Hz(48k)라 ADC 자체는 문제없음. 즉 저역 손실은 **외부
  커플링 캡에서만** 생길 수 있으니 거기만 보면 됨.

## 4. 브링업 순서 (각 단계 검증 후 다음)

### Step 0 — 사전점검
- [ ] 모듈 핀헤더 납땜 상태 확인(미납이면 먼저 납땜)
- [ ] MP1584 단독으로 **5.0V** 세팅 (위 ⚠)
- [ ] PCM1808 모듈 모드 점퍼: **ESP32가 I2S 마스터 → PCM1808는 슬레이브**, FMT=I2S(24bit).
      (모듈이 마스터로 하드와이어돼 있으면 클럭 충돌 → 반드시 확인)

### Step 1 — 디스플레이 스모크 테스트 (USB만)
- `display_bringup.c` 하단 주석의 임시 `app_main`(HELLO ST7796) 플래시.
- [ ] 백라이트 켜짐 + "HELLO ST7796" 표시 → SPI·핀·디스플레이 OK.
- 색 반전/RGB 뒤바뀜이면 `esp_lcd_panel_invert_color` / `rgb_ele_order` 조정.

### Step 2 — 전체 펌웨어 (USB)
- 정식 빌드 플래시. 런처("PEDAL DISPLAY" + 앱 메뉴) 표시 기대.
- [ ] 상/하(또는 좌/우)로 하이라이트 이동, 확인(OK)으로 앱 진입
- [ ] 방향키 길게 누르면 오토리피트(빠른 반복 이동)
- [ ] 풋스위치 숏 → 다음 앱 순환 / 롱 → 튜너 점프 (입력 없어도 "—" 표시되면 정상)
- [ ] 홈 숏 → 공통 팝업(첫 항목 Exit), 확인으로 홈 복귀 / 홈 롱 → 즉시 홈 복귀
- 참고: 뮤트(GPIO21)는 아직 미배선이라 튜너 점프 시 GPIO만 토글되고 소리 변화는 없다(정상).

### Step 3 — 전원 레일
- ELB040202→MP1584(5V검증)→DevKit 5V, MP1584→AMS1117→3.3V 배선.
- [ ] 부하 연결 **전** 멀티미터로 5.0V / 3.3V 확인
- [ ] 모든 GND 공통 결선
- [ ] DevKit이 5V 핀만으로 부팅(USB 분리) 되는지

### Step 4 — PCM1808 (I2S)
- 디지털 3.3V + 아날로그는 AMS1117 클린 3.3V. I2S 4선(15/16/17/18) 결선.
- [ ] 입력 무신호 상태로 Sound Monitor 진입 → 바닥 노이즈/평탄 스펙트럼 보이면 I2S OK
- I2S overflow 로그가 계속 뜨면 클럭/슬롯 설정 재확인.

### Step 5 — 아날로그 + 실신호
- TL072 가상GND(4.5V 측정) → 버퍼/게인 → AC커플 → PCM1808 IN.
- [ ] 기타 연결 → **Tuner** 화면에서 음정 검출(센트 니들 반응)
- [ ] **Sound Monitor**에서 스펙트럼 반응
- [ ] **저역 확장 확인** (이번 튜너 업데이트의 핵심): 베이스 저E(~41Hz)·드롭D(~73Hz),
      가능하면 5현 저B(~31Hz)까지 **음 이름이 안정적으로 뜨는지**. 저역에서 음이 흔들리거나
      한 옥타브 위로 튀면 → AC커플 캡(저역 손실) 또는 게인/노이즈 확인.
- 클리핑(파형 깨짐/과반응)이면 게인 ↓(Rf 낮춤).
- 참고: 튜너 갱신 주기 24ms(약 42Hz) — 저역은 원래 천천히 맞추므로 체감 무방.

### Step 6 — (J201 도착 후) 출력 + 뮤트
- 별도 단계. 아래 보류 항목 참고.

## 5. 보류 항목 (지금 조립 안 함)

- **J201 뮤트 (GPIO21)**: 알리 배송이라 늦게 도착. 그리고 J201은 공핍형이라
  **게이트를 음전압으로 꺼야** 하는데 ESP32 GPIO는 0~3.3V(양수)만 낸다. 이 게이트
  드라이브(+RC 소프트램프)를 J201 실물 도착 시점에 따로 설계한다. 뮤트 없이도
  풋스위치 튜너 점프·튜너·비주얼라이저는 전부 동작하므로 첫 조립엔 지장 없음.
- **SD 카드**: 펌웨어의 SD 마운트(`esp_vfs_fat_sdspi_mount`)가 아직 미구현 스텁.
  `content_fs_register`는 `/sdcard` 마운트를 가정만 함. SD 핀도 펌웨어에 미정의 →
  마운트 구현할 때 핀 배정 후 배선. 이미지 화면은 그 전까지 플레이스홀더.
- **MIDI (Phase 2)**: 파서·매핑은 완료, UART RX + 옵토 절연 회로 미연결. 옵토커플러
  (6N138/H11L1)·TRS-MIDI 잭은 Phase 2 부품.

## 6. 빠른 트러블슈팅

| 증상 | 1순위 확인 |
|------|-----------|
| 화면 안 켜짐 | 백라이트 GPIO8, VCC 3.3V, RST/CS/DC 배선 |
| 화면 색 이상 | invert_color / rgb_ele_order(BGR↔RGB) |
| Monitor 무반응 | I2S 4선, PCM1808 슬레이브 모드/FMT, 아날로그 전원 |
| Tuner 검출 안 됨 | TL072 Vref 4.5V, 게인, AC커플 캡, 입력 바이어스 |
| 노이즈 과다 | 스타 그라운드, 입력 점퍼 최단화, 아날로그/디지털 레일 분리 |
| 보드 재부팅 반복 | 전원 부족(MP1584 전류/발열), 5V 강하 확인 |
| DevKit 사망 | MP1584 출력이 5V 초과였는지 (사전 세팅 누락) |
| 버튼 반응 이상 | GPIO 1·2·4·5·6·13(손버튼)·7(풋스위치) → GND 결선, 풀업 enable |
